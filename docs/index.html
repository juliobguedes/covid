COVID-19 Timeline Overview

Bem vindo a Timeline de Overview do COVID-19. O objetivo principal desta página é transmitir uma visão geral de como a doença se espalhou pelo mundo a medida que o tempo passa. Recomendo que veja esta página num browser web, num computador, já que não foi criada com a intenção de ser responsiva, e por isso pode não funcionar como esperado em disposítivos móveis. O objetivo é apenas transmitir a informação contida.

Os dados utilizados nas visualizações foram obtidos pelo site e pelo repositório no GitHub da universidade Johns Hopkins, nos Estados Unidos. O código que gera os gráficos e o mapa estão disponíveis neste repositório do GitHub. Sinta-se livre e encorajado para sugerir adições, modificações, abrir issues e pull requests no repositório.

No momento, os dados desta página estão sendo atualizados manualmente todos os dias, utilizando os scripts contidos no repositório. Dessa forma, é possível que caso acesse pouco após meia noite ou durante a madrugada, a página ainda não esteja atualizada com os dados do dia anterior.

Welcome to the COVID-19 Timeline Overview. The main objective of this page is to give an overview look about how the disease spread in the world as the time passes. I recommend you to see this page in a web browser: it was not made to be responsive, which means that it might not work as intended in mobile devices. The exclusive goal here is to focus on the information here contained.

The data used to generated the visualizations were obtained from the Johns Hopkins University website and GitHub repository. The code that generates the plots and the map are available in my GitHub repository. Feel free and encouraged to suggest additions, modifications, open issues and pull requests.

So far, the data is being manually updated every day, using the scripts in the repository. This considered, its possible that the data is outdated, which means that it will not contain information about the day before you are seeing it.

O site ainda está em desenvolvimento, o que pode ser visto caso você acesse os notebooks contidos no repositório. Assim, fique a vontade para contribuir e melhorar o conteúdo, incluindo informações sobre a doença e regiões infectadas.

The website is still in development and that can be seen if you access the notebooks in the repository. Therefore, feel free to contribute and improve the contents, including information about the disease and infected regions.
